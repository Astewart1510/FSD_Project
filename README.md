# Installation

## Setup

1. Clone this repository

2. Move into this `financials` subfolder:

	```bash
	cd financials
	```

3. Install package dependencies:

	```bash
	npm install
	```

# Run Dashboard
Once packages are installed:

	```bash
	npm run serve
	```
